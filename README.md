#Oodle-api

Make any string full of oodles!

##Installation

Simply install globally using `npm i -g oodles-api` and run with `oodles-api`.

##Usage

By default will use port 3000, but this can be overridden using the command-line option `--port`.

##Test

```shell
npm test
```
